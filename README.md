# answer-for-task1

## Задача 1. Сбор и обработка данных о парковках Алматы через API 2ГИС

### Описание задания

Задание было выполнено в рамках тестового задания на стажировку.

Скрипт собирает данные о парковках Алматы через API 2GIS и сохраняет результаты в форматах Excel и CSV.

### Собираются следующие данные:

* ID объекта;
* название парковки;
* адрес;
* координаты;
* ссылка на объект в 2GIS;
* статус (платная / бесплатная / неизвестно);
* тариф;
* количество мест;
* тип парковки;
* связанный объект;
* часы работы.

### Дополнительно выполняется:

* поиск по нескольким районам Алматы;
* поиск по нескольким ключевым запросам;
* удаление дубликатов;
* заполнение отсутствующих значений статусом «Неизвестно».

## Как запустить задание

### 1. Скачать репозиторий

```bash
git clone https://github.com/comepotyyy/task1_2gis.git
```

### 2. Перейти в папку проекта

```bash
cd task1_2gis
```

### 3. Установить необходимые библиотеки

```bash
pip install pandas requests openpyxl tqdm
```

### 4. Запустить Jupyter Notebook

Если используете Jupyter Notebook:

```bash
jupyter notebook
```

Если используете JupyterLab:

```bash
jupyter lab
```

### 5. Открыть файл ноутбука

```
task1_2gis.ipynb
```

### 6. Выполнить код

Запустите все ячейки ноутбука последовательно сверху вниз.
